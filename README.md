================================================================================
                    ПРОГРАММА: МИНИМАЛЬНОЕ ЧИСЛО ИЗ ЕДИНИЦ
================================================================================

ОПИСАНИЕ
--------------------------------------------------------------------------------
Данная программа находит минимальное положительное число, состоящее только из 
цифр '1', которое делится на заданное целое число n без остатка.

Например:
- Для n = 3 ответ: 111 (три единицы, 111 / 3 = 37)
- Для n = 7 ответ: 111111 (шесть единиц, 111111 / 7 = 15873)
- Для n = 12 ответ: NO (числа из единиц не делятся на чётные числа и на 5)

АЛГОРИТМ
--------------------------------------------------------------------------------
Программа использует математический подход с отслеживанием остатков:

1. Последовательно строим числа из единиц: 1, 11, 111, 1111, ...
2. На каждом шаге вычисляем остаток от деления на n
3. Если остаток стал 0 — найдено искомое число
4. Если остаток повторился — такого числа не существует (цикл)

ВХОДНЫЕ ДАННЫЕ
--------------------------------------------------------------------------------
- Одно целое число n (1 ≤ n ≤ 10^6)
- Число не должно делиться на 2 или на 5 (иначе ответ "NO")

ВЫХОДНЫЕ ДАННЫЕ
--------------------------------------------------------------------------------
- Строка из цифр '1' — минимальное число, делящееся на n
- Или "NO" — если такого числа не существует

ОШИБКИ ВВОДА
--------------------------------------------------------------------------------
Программа проверяет корректность ввода:
- Пустая строка → "Error: Input cannot be empty."
- Не число → "Error: Please enter a valid integer."

ПРИМЕРЫ РАБОТЫ
--------------------------------------------------------------------------------
Ввод:  3
Вывод: 111

Ввод:  7
Вывод: 111111

Ввод:  12
Вывод: NO

Ввод:  (пустая строка)
Вывод: Error: Input cannot be empty.

Ввод:  abc
Вывод: Error: Please enter a valid integer.
